Resumen del Proyecto

Las respuestas de la Parte #3 se encuentran en el Readme del repositorio

Aplicación web construida con Angular V20 que permite a los usuarios crear, editar y eliminar Productos y asociarlos con Categorias.

Stack Tecnológico
  • • Angular 20 con signals
  • • Abstinencia de ZoneJs
  • • TypeScript
  • • Tailwind CSS
  • • NgPrime
  • • RxJS/NgRx
Características Clave
  • • Uso de Store con Signals
  • • Operaciones CRUD
  • • Integración con Back
  • • Api Rest
  • • Validación de datos
